## Quick Start
#### Download
~~~
git clone https://github.com/ZG-frank/react-start-toolkit.git
~~~

#### Install
~~~
yarn install
~~~

#### Run
~~~
npm run dev
~~~

#### Build
~~~
npm run build
~~~

#### Server
~~~
npm run server
因为采用了BrowserRouter，出于浏览器安全的原因，build之后的文件需要启动服务器来访问
这里使用了express来启用服务器，默认是8090端口并返回dist文件夹中的index.html
~~~
